# LLM Learner

这个仓库用于系统化整理和记录我在大语言模型（LLM）方向的学习过程。

## 1. 学习目标

- 建立完整的 LLM 知识框架（概念 -> 原理 -> 实践）
- 沉淀可复用的实验与工程经验
- 形成持续更新的学习笔记与资源库

## 2. 知识地图

计划按以下主题逐步完善内容：

- 基础概念：Transformer、Token、Embedding、注意力机制
- 训练与微调：预训练、SFT、RLHF、对齐
- 推理与工程：Prompt Engineering、RAG、Agent、评测
- 部署与优化：量化、蒸馏、推理加速、成本控制
- 安全与治理：幻觉、越狱、隐私与合规
